BMI Calculator

A stunning, fully responsive Body Mass Index (BMI) calculator featuring a 3D interactive background, glassmorphism UI design, and real-time health visualizations. Built with HTML5, Bootstrap 5, and Three.js.

Fully Responsive: Mobile-first design that looks great on phones, tablets, and desktops.

Single File Deployment: All logic, styling, and assets are contained in a single index.html file for easy sharing and hosting.
